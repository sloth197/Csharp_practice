# Csharp_practice

<h2>1. 간단한 채팅 프로그램</h2>
- C#을 사용하여 Networking, Threading, User Interface(UI) 구현<br/>
- 클라이언트는 서버의 IP주소와 포트를 지정, 연결 후 메세지를 보내고 서버로부터 메세지를 수신하는 역할을 합니다. <br/>
    사용자가 콘솔에 입력한 메세지를 서버로 전송하면 별도의 Thread를 통해 서버로부터 수신되는 메세지를 처리합니다.
